# vin-skills

Claude Code skills collection.

## Skills

| Skill | Description |
|-------|-------------|
| [vin-github-star-search](./vin-github-star-search/) | Search GitHub starred repositories using `gh` CLI |
| [vin-github-issue-search](./vin-github-issue-search/) | Search GitHub issues in any repository using `gh` CLI |
| [vin-code-review](./vin-code-review/) | Gemini Code Assist automated review-fix workflow |

## Installation

```bash
claude install-skill ByteObject/vin-skills/vin-github-star-search
claude install-skill ByteObject/vin-skills/vin-github-issue-search
claude install-skill ByteObject/vin-skills/vin-code-review
```
